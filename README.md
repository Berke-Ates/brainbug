# BrainBug
Online Brainfuck Debugger 

## Features
- Code segment
- Stepper (with variable stepsize)
- Runner with different speeds
- Run, Pause, Reset
- \# breakpoints
- // Code comments
- Memory view in ASCII + RAW
- Dark theme
- Input / Output
- Memsize
- Cellsize
- Cell wrapping
- ASCII Table
- Runtime edit 
- Memory edit
